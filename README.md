#Pico Held 2 hardware resources

This site is all about the Pico Held 2's hardware: schematics, 3D design files, gerber files for PCB production, component specifications and so on.

## Obtaining the hardware as a kit

It's probably the cheapest solution if you buy the components from me. I bought them in bulk to save on shipping costs. I sell a "kit" on eBay. It contains the mainboard (including the MCU), shoulder button PCBs, LCD, rubber pads, screws, speaker, analog stick, and buttons. However, you will still need the 3D-printed parts (case, start/select buttons, and shoulder buttons) and the battery.
Since I'm not running a business and don't intend to, if you want to buy a kit, drop me an email and I'll create an offer on eBay and send you the link (if I can ship to your country that is). The kit costs around 80 Euros. You'll find my email address in the header of the main file.
