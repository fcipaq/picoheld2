These are the gerber files, schematics and BOM and CPL files for the mainboard (ESP32-P4 version)

!!! When manually installing the ESP32-P4-SOM, pay close attention to its orientation. Some versions have a dot imprint on the metal shell that can be misleading because it does NOT indicate pin 0.

The PCB thickness is 1.6 mm

The BOM and CPL files are in xlsx format ready to be manufactured at JLCPCB. I already checked the alignment. Unfortunately, some components are unavailable in JLC's parts library.

The JPG image is for reference only and might not accurately reflect the current state.
IMPORTANT: The orientation of the SOM is correctly depicted in the image. Counterintuitively, the circular dent in the SOM's cover must not be aligned with the dot on the PCB.
