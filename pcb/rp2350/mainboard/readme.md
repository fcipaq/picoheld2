These are the gerber files, schematics, BOM and CPL files for the mainboard (RP2350 version)

The PCB thickness is 1.6mm

I recommend using PCBWay's "shared project" as you don't have to deal with any of these files and you just have to order them.

The BOM and CPL files come in two versions:

a) generic
b) for JLCPCB (in an xlsx format). I already checked the alignment. Unfortunately, some components are unavailable in JLC's parts library.
